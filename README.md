# Dokumentace
Centrální repozitář dokumentace BI oddělení společnosti Euromedia Group, a.s. Slouží k ukládání, správě a sdílení technické i procesní dokumentace.

Tento repozitář slouží jako centrální úložiště dokumentace BI oddělení společnosti Euromedia Group, a.s.

## Struktura repozitáře

- **Dokumentace k reportům**  
  Obsahuje popis BI reportů, jejich účel, zdroje dat a logiku výpočtů.

- **Dokumentace k databázi**  
  Obsahuje dokumentaci databázových struktur, tabulek, pohledů a datových vazeb.

Cílem je mít přehlednou, aktuální a snadno dostupnou dokumentaci pro celý BI tým.
